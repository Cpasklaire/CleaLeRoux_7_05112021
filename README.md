Bienvenue sur Grouposocial le réseau social de Groupomania

## Installer le projet
Clonnez le github
Dans la racide du projet lancer
```
npm install
```
Créé la database dans le clientMySQL avec le fichier database.sql
Modifier le .env

### Lancer l'appli
Dans la racine du fichier lancer
```
npm run frontstart
npm run backstart
```
